# Music Addon Features

## Core Playback
- Play from YouTube/Spotify URLs or search
- Pause, resume, skip, stop playback
- Volume control (0-100%)
- Loop mode for current song
- Seek to timestamp (MM:SS or seconds)

## Queue & History
- View queue with positions
- Shuffle, clear queue
- Queue persistence (save/restore)
- Preemptive download for seamless playback
- Duplicate detection with warnings
- Play history tracking (50 songs)
- Max queue size & song duration limits

## Spotify Integration
- Play tracks, playlists, and albums
- Automatic YouTube conversion

## User Features
- Rich now playing with auto-updating progress bar █████████░░░░░
- Interactive search (7 results)
- Personal volume preferences
- Server & personal statistics
- Top songs leaderboard
- Personal favorites list
- Custom playlists (10 per user, 100 songs each)

## Audio Effects (12 total)
- Nightcore, Bass Boost, 8D Audio, Vaporwave
- Treble, Echo, Reverb, Chipmunk
- Deep Voice, Distortion, Tremolo, Vibrato

## Advanced
- Radio mode (auto-play similar songs)
- Lyrics display (Genius API)
- Smart cache with popularity tracking
- Connection recovery with auto-reconnect
- Rate limiting
- Whitelist/Blacklist system
- Admin roles with special permissions

## Technical
- Age-restricted & unavailable video handling
- Retry logic (3 attempts)
- Auto cleanup temporary files
- Voice channel validation
- Permission checks
- Configurable file size & duration limits

## Configuration
- Feature flags for enable/disable
- Customizable messages & colors
- Rename commands to avoid conflicts
- Custom timeouts & storage paths
